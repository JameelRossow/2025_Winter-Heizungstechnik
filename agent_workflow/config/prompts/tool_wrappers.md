# Tool Wrapper Hinweise

- `DocReader` liefert Abschnitte aus dem aktuellen Kapitel oder benachbarten Kapiteln.
- `WebSearch` darf nur genutzt werden, um Normen/Regelwerke zu zitieren; Quellen bitte als Markdown-Link hinterlegen.
- Achte auf kurze Tool-Ein-/Ausgaben (< 2000 Token), damit der Loop schlank bleibt.
