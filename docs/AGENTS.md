# Agent Guidelines - Docs

- Nenne ich nur eine Kapitelnummer (z. B. `5.4` oder `5.4_.md`), ohne den vollständigen Dateinamen, dann gehe ich davon aus, dass die zu bearbeitende Datei noch offen ist unter `docs/`. In diesem Fall liste ich zuerst den gesamten Inhalt von `03_Dokumentation/docs` auf, damit die gewünschte Markdown-Datei eindeutig identifiziert werden kann, bevor ich fortfahre.
