# 4.3 Verteilerstruktur / Steigestränge / Unterverteilung

**Technikzentrale:** Pufferspeicher als hydraulischer Knoten, von dem aus alle Hauptabgänge geführt werden.

- HK-Verteilung: Aufteilung auf sechs Heizkörper-Stränge inklusive Armaturen zur Bereichstrennung (Betrieb/Wartung). Siehe Schnitt in Anlage 8.1.
- FBH-Verteilung: Gemischter Kreis mit Steigesträngen und Etagen-Heizkreisverteilern (Roth). Siehe Anlage 8.14.
